## Getting Started

First, install the dependencies:

```bash
pnpm install
# or
npm install
# or
yarn install
```

Then, run the development server:

```bash
pnpm dev
# or
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deploy

### Coolify (Recomendado)
1. Crea servicio Git en Coolify desde este repo.
2. Env vars desde .env.example (Supabase URL/key).
3. Puerto 3000. Auto-detectará Dockerfile.
4. Supabase externo: Ejecuta migrations `supabase/*.sql`.

### Local test
```bash
cp .env.example .env
docker compose up --build
```
